Hugo

Le sujet du jour : comment créer son blog avec Hugo, un générateur open-source de sites Web statiques. C’est avec cet outil que j’ai construit ce blog. Je ne vais pas détailler ici toutes les étapes de création et de personalisation car les tutos Hugo et articles de blog sur le sujet expliquent très bien les différentes étapes. Vous trouverez donc ci-dessous les différents éléments que j’ai implémenté sur ce blog avec les liens correspondants et si nécessaire quelques explications complémentaires sur certains points.
